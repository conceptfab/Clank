# Clank

Clank is a tiny free macOS menu-bar app for Apple Silicon Macs. It plays short voice reactions when you tap or smack the laptop, and it can play a door-slam sound when the lid moves.

## Download

- [Download the latest Clank release](https://github.com/conceptfab/Clank/releases/latest)
- [View the source on GitHub](https://github.com/conceptfab/Clank/releases)
- [Read the install guide](https://github.com/conceptfab/Clank/releases)

## Features

- Smack mode with ten short voice reactions.
- Lid mode with a door-slam sound.
- Menu-bar controls with pause, sensitivity, and custom sounds.
- No analytics, no tracking, no newsletter, no paid tier.

## Install

1. Download the DMG from the latest GitHub release.
2. Drag Clank to Applications.
3. Launch the app and approve the native helper install prompt.
4. If Gatekeeper warns on first launch, right-click Clank, choose Open, and confirm once.
