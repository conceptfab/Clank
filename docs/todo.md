User autoryzuje raz przy pierwszym setupie (przez System Settings → Login Items lub przez prompt z app), helper od tamtej pory startuje jako root przy każdym boocie, UI gada z nim przez XPC. Brak haseł przy kolejnych uruchomieniach.
